# Gpt-Test01

This repository contains a simple HTML5 game called **Plane Battle Game**.

To play, open `index.html` in a web browser and use the arrow keys to move your
plane and the spacebar to shoot. Enemies will spawn from the top of the screen.
Try to destroy them and avoid collisions.

Enter the cheat code `BULLET` during play to unlock weapon switching. A drop-
down selector will appear that lets you choose between Normal, Heavy or Triple
bullets.
